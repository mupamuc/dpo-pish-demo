# ДПО ПИШ СПбПУ — демо-прототип

**Живая демка:** https://mupamuc.github.io/dpo-pish-demo/

Прототип новой страницы «Дополнительное профессиональное образование» для сайта
ПИШ СПбПУ «Цифровой инжиниринг» (pish.spbstu.ru/courses/additional).

## Что внутри

- **Каталог программ** — 4 типа продукта: курсы повышения квалификации,
  президентская программа, симуляторы и тренажёры, онлайн-курсы.
  Сайдбар-фильтр: аудитория (частным лицам / компаниям), поиск, 6 групп
  направлений, формат, длительность.
- **Страница программы** — вкладки «Описание»/«Программа», блок фактов,
  обе цены сразу (частному лицу и компании), похожие программы.
- **Оценка компетенций через симуляторы** (`#assessment`) — профиль по 6 осям
  из телеметрии, зачёт по гейтам, ИИ-диагност потерь.
- **Подбор программ для HR** (`competencies.html`) — роль сотрудника или набор
  компетенций → ранжированная подборка; матрица «компетенции × программы».

## Ключевое правило

Сайт **не собирает данные посетителей**: ни форм, ни кнопки «Записаться».
Все действия ведут к контактам (почта/телефон) — «Связаться с нами».
Подбор программ работает полностью на клиенте.

## Дисклеймер

Это **демо-прототип для обсуждения**, не рабочий сайт.
Все ФИО, e-mail, телефоны и цифры — **вымышленные или обезличенные
плейсхолдеры**. Маппинг «программа ↔ компетенции» — черновая разметка,
не подтверждена руководителями программ.

## Запуск локально

Статические файлы, сборки нет:

```
python -m http.server 8000
# → http://localhost:8000/
```
